# Dotfiles
This repository contains configuration for:
- bash
- zsh
- vim
- nvim
- i3
- polybar
- [picom](https://github.com/jonaburg/picom)
- openbox
- alacritty

## Key bindings

| Key                                | Action        |
| ---------------------------------- | ------------- |
| <kbd>Super</kbd>+<kbd>Return</kbd> | Open terminal |
