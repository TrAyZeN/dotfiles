# Dotfiles
This repository contains configuration for:
- bash
- zsh
- vim
- nvim
- i3
- polybar
- [picom](https://github.com/jonaburg/picom)
- openbox
- alacritty
- kitty
- rofi

My bash and vim configs are focused on minimalism.
